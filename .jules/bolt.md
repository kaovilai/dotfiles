## 2024-05-18 - Fast file age checking with native Zsh extended globbing
**Learning:** Using `stat -f "%m"` or `stat -c "%Y"` combined with `date +%s` spawns multiple external subprocesses, which is extremely slow when checking file age frequently in Zsh (e.g. for caching). Native Zsh extended globbing qualifiers `(#qN.ms+seconds)` can accomplish the same file age check in a fraction of a millisecond without any subshells or subprocesses.
**Action:** Replace `stat` and `date` with `(#qN.ms+seconds)` for file age checks in Zsh scripts. When doing this inside functions that might be called with different options, ensure `setopt local_options extended_glob` is used, and store the result in an array to properly check if the pattern matched (as bare patterns evaluate to literal strings if they don't match, or error if `null_glob` isn't set, while `N` makes them empty arrays).
